# Attendance

本工程是在自己大学阶段的校级项目成果上，进行部分功能的删减修改及代码重构后完成的。项目主要是起到练手作用，采用了MVP架构，遵循Material Design的设计理念，并使用了RxJava、RxAndroid和ButterKnife等开源框架，附带[Java版本](https://github.com/lpq29743/Attendance)，欢迎star和提issue和PR，共同学习，共同进步！
